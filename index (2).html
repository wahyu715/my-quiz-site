情報Ⅰ - WWWとURL

ア __________ は、インターネットで不特定多数の人に情報を発信する通信サービスである。

WWWでは イ __________ というプロトコルが使われ、

ウ __________ と呼ばれる単位で情報が発信される。

ウェブページは エ __________ という言語で書かれ、文章、音声、画像、動画などの情報を統合することができる。

また、 オ __________ （リンク）という機能で、さまざまな関連付きができ、簡単に移動できる。

Webページを見るためのソフトウェアを カ __________ （ブラウザ）という。

インターネット上のWebページの住所を示すものを キ __________ という。

コンピュータネットワークを使って、手紙のようなメッセージを送受信するシステムを ク __________ という。

それを送受信するソフトウェアを ケ __________ という。

電子メールの宛先を コ __________ という。

電子メールの受け渡しを行うサーバーを サ __________ という。

電子メールを送信するときは シ __________ 、受信するときは ス __________ や セ __________ というプロトコルが使われる。

答え

ア: WWW

イ: HTTP

ウ: ウェブページ

エ: HTML

オ: ハイパーリンク

カ: ウェブブラウザ

キ: URL

ク: 電子メール

ケ: メーラー

コ: メールアドレス

サ: メールサーバー

シ: SMTP

ス: POP

セ: IMAP
